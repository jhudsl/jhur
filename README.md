# jhur Package:

The goal of `jhur` is to provide accompanying functions for the
Introduction to R class at Johns Hopkins.

## Installation

You can install `jhur` from GitHub with:

``` r
# install.packages("remotes")
remotes::install_github("jhudsl/jhur")
```
